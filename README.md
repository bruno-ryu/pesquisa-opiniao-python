# Pesquisa de Opinião - TudoWeb

Este projeto foi desenvolvido em Python para coletar opiniões de clientes sobre o atendimento.

## Funcionalidades
- Coleta nome, idade e opinião
- Classificação:
  - 1: Excelente
  - 2: Bom
  - 3: Ruim

## Resultado
Exibe:
- Quantidade de respostas "Excelente"
- Quantidade de respostas "Ruim"

## Observação
Foram realizados testes com 10 entrevistados para validação do sistema, conforme solicitado.
